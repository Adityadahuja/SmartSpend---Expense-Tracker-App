# Requirements Gathering

## Stakeholder Interview

**Interviewee**: College Student (Target User)
**Interviewer**: Team SmartSpend
**Date**: 2023-10-15

**Q1: How do you currently track your expenses?**
*A1: I mostly use my bank app or just try to remember. Sometimes I write it down in a notebook, but I often forget.*

**Q2: What is the biggest problem you face with current methods?**
*A2: It's hard to see where my money is actually going. I know I spent it, but I don't know if it was on food, travel, or entertainment.*

**Q3: Would you prefer a mobile app or a web app?**
*A3: A web app that works well on mobile would be great. I don't want to install heavy apps.*

**Q4: Is privacy a concern for you?**
*A4: Yes, definitely. I don't want my financial data shared with third parties or advertisers.*

**Q5: What features are "must-haves" for you?**
*A5: Quick entry of expenses, seeing a monthly summary, and maybe some charts. Also, distinguishing between income and expense is important.*

## Questionnaire

**Survey Target**: 50 College Students

**1. Do you track your daily expenses?**
- [ ] Yes, regularly (20%)
- [ ] Sometimes (50%)
- [ ] No (30%)

**2. What tools do you use?**
- [ ] Excel/Spreadsheet (15%)
- [ ] Mobile Apps (40%)
- [ ] Notebook/Paper (25%)
- [ ] None (20%)

**3. What features do you value most? (Select top 3)**
- [x] Ease of use (80%)
- [x] Data Visualization/Charts (60%)
- [x] Privacy/Security (50%)
- [ ] Budgeting alerts (40%)
- [ ] Export data (30%)

**4. Would you use an app that stores data locally without requiring an account?**
- [ ] Yes (70%)
- [ ] No (10%)
- [ ] Maybe (20%)

**Conclusion**:
The survey indicates a strong need for a simple, private, and visual expense tracking tool. Users prioritize ease of use and data visualization over complex budgeting features.
